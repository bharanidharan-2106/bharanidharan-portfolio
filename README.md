# Bharani-Portfolio

Personal Portfolio Website built with **React** and **Vite**.

## Live Website
https://bharani-dev.vercel.app/

## ✨ Features
- **Silky Transitions**: Custom scanline and sweep effects.
- **Email Integration**: Live contact form powered by EmailJS.
- **Modern UI**: Full-stack developer theme with floating code symbols and glassmorphism.
- **Premium Aesthetics**: Subtle gradients and animated section dividers.

## 🚀 Tech Stack
- Frontend: React.js
- Styling: Vanilla CSS
- Animations: Custom CSS3 Animations
- Icons: React Icons
- Deployment: Vercel (Ready)
