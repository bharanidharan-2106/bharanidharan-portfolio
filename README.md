# Bharanidharan Portfolio

A modern, high-performance personal portfolio website designed to showcase backend expertise, full-stack projects, and technical skills. The application emphasizes clean UI design, smooth user experience, and production-ready frontend practices.

Live Demo: https://bharanidharan-portfolio.vercel.app/

---

## Overview

This portfolio serves as a central platform to present projects, technical skills, and professional profile in a structured and interactive manner. It is built with a focus on performance, responsiveness, and user engagement.

The application follows modern frontend development practices with optimized builds, reusable components, and clean styling architecture.

---

## Key Features

* Developer-focused UI with structured layout and clean visual hierarchy
* Smooth animations and transitions for enhanced user experience
* Interactive components including typewriter effects and section-based navigation
* Integrated contact system using EmailJS for direct message delivery
* Copy-to-clipboard functionality for quick access to contact details
* Fully responsive design across desktop, tablet, and mobile devices

## Setup and Installation

### 1. Clone the Repository

```bash
git clone https://github.com/bharanidharan-2106/bharanidharan-portfolio.git
cd bharanidharan-portfolio
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

Note:
Environment variables are not included in the repository for security reasons.

---

### 4. Run the Application

```bash
npm run dev
```
